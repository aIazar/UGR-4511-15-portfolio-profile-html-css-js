Countdown Timer

The Countdown Timer Application is a user-friendly Java-based program designed to help users easily set a countdown to a specific date and time. This graphical user interface (GUI) application is developed using the Swing framework. This Countdown Timer Application can be particularly useful for users who need to track time for various events, such as deadlines, special occasions, or timed tasks. Minesweeper Game

The Minesweeper Game, implemented in Java with a Swing-based GUI, challenges players to reveal all non-mine cells on a 10x10 grid while avoiding mines. Players can flag suspected mines, and adjacent mine counts guide their moves. Clicking a cell with zero adjacent mines reveals nearby cells automatically. The game ends when a mine is revealed or all safe cells are uncovered. A "Play Again" button allows restarting, and clear instructions are provided at the top.
